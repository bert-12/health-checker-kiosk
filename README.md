Health Self-Check Kiosk

A web-based BMI (Body Mass Index) calculator and health self-check kiosk that helps users monitor their health status and stores data automatically in Google Sheets.

About The Project

The Health Self-Check Kiosk is a simple, user-friendly web application that allows users to calculate their Body Mass Index (BMI) based on their weight and height. It provides instant feedback on their health status with personalized recommendations. The application also automatically saves all submissions to Google Sheets for record-keeping and tracking purposes.

Problem It Solves

1. Lack of Health Awareness
Many people are unaware of their BMI and what it means for their health. This app provides an easy way to calculate BMI and understand its implications.

2. Manual Record Keeping
Tracking health data manually can be tedious and error-prone. This app automatically saves all submissions to Google Sheets, making it easy to track health progress over time.

3. Accessibility
The app is web-based and responsive, making it accessible from any device with an internet connection.

4. Immediate Feedback
Users get instant results and personalized health recommendations, encouraging them to take action on their health.

5. Data Collection for Health Monitoring
Healthcare providers or individuals can use the Google Sheets integration to monitor health trends over time.

Features

BMI Calculation - Accurate BMI calculation based on weight and height
Health Categories - Classifies BMI into Underweight, Normal, Overweight, and Obese
Personalized Recommendations - Provides health tips based on BMI category
Google Sheets Integration - Automatically saves all submissions to Google Sheets
Form Validation - Ensures all required fields are filled correctly
Reset Functionality - Easy form reset with one click
Responsive Design - Works on desktop, tablet, and mobile devices
User-Friendly Interface - Clean and intuitive design
Real-Time Results - Instant BMI calculation and display

Technologies Used
- Frontend: HTML5, CSS3, JavaScript (Vanilla)
- Backend: Google Apps Script
- Database: Google Sheets
- Version Control: Git & GitHub
- Deployment: GitHub Pages
